3.4 IDE集成

界面 第三章：界面操作

OpenCode提供与VS Code、Cursor等IDE的无缝集成，通过终端访问实现。

支持的IDE

  • VS Code
  • Cursor
  • Windsurf
  • VSCodium
  • 其他兼容VS Code的编辑器

安装扩展

在IDE的集成终端中运行 opencode，扩展会自动安装。也可以在扩展市场搜索"OpenCode"手动安装。

快捷键

功能macOSWindows/Linux
打开/聚焦OpenCodeCmd+EscCtrl+Esc
新建会话Cmd+Shift+EscCtrl+Shift+Esc
插入文件引用Cmd+Option+KAlt+Ctrl+K

上下文共享

扩展自动将当前文件选择或活动标签页共享给OpenCode。使用文件引用语法：

@File#L37-42  # 引用37-42行

功能特性

  • 分屏终端：OpenCode在分屏终端中打开
  • 上下文感知：自动获取当前文件信息
  • 快速启动：一键打开或聚焦
  • 会话管理：支持多会话切换

故障排除

如果自动安装失败：

  1. 确保在IDE的集成终端中运行 opencode
  2. 验证IDE的CLI工具已安装（code、cursor等）
  3. 通过命令面板安装shell命令
  4. 确认扩展安装权限已启用
